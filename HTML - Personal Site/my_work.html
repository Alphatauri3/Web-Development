System on Module (SoM)

System on Module (SoM), is a single physical module that incorporates many common computing components. Common CoM components include a CPU, RAM, flash memory, a power management unit, an Ethernet bus, Wi-Fi and Bluetooth radios. The availability of all this hardware as a single unit reduces the complexity of final product designs.
